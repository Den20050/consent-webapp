Политика конфиденциальности и соглашение об обработке персональных данных

(соответствует требованиям 152-ФЗ «О персональных данных»)

1. Общие положения

  1. Настоящий документ определяет порядок обработки персональных данных пользователей бота QRConnect (далее — «Бот»), разработанного в мессенджере Telegram.
  2. Использование Бота означает безусловное согласие пользователя с условиями настоящей Политики.
  3. Все данные обрабатываются в соответствии с 152-ФЗ, GDPR (для пользователей из ЕС) и внутренними регламентами оператора.

2. Какие данные собираются

Бот обрабатывает следующие персональные данные:

  • Имя (только буквы, до 15 символов)
  • Возраст (от 18 до 99 лет)
  • Цели знакомства (выбор из предложенных вариантов)
  • Контактные данные: телефон или @username Telegram
  • Произвольный текст (до 1000 символов, например, ссылки на соцсети)
  • Фото/видео (форматы: JPEG, PNG, MP4, MOV; длительность видео — до 60 секунд)
  • Технические данные: IP-адрес, тип устройства (iOS/Android), логи действий

3. Цели обработки данных

  1. Данные используются для:
    • Создания анкет-визиток
    • Передачи анкет владельцам профилей (третьим лицам) только после согласия пользователя
    • Анализа статистики (устройства, цели знакомства, переходы по рекламе)
    • Рассылки рекламных материалов (если пользователь не отказался)
  2. Файлы (фото/видео) хранятся временно: после отправки анкеты удаляются с сервера, остаются только в чате получателя.

4. Хранение и защита данных

  1. Срок хранения:
    • Логи действий — 8 месяцев (после архивации)
    • Анкеты пользователей — до момента удаления владельцем профиля или администратором
  2. Данные защищаются с помощью:
    • Шифрования передаваемой информации (HTTPS, TLS)
    • Ограниченного доступа (только администратор и владелец анкеты)

5. Передача третьим лицам

  1. Данные передаются только:
    • Владельцу анкеты-визитки, с которой взаимодействовал пользователь
    • Рекламным партнерам (анонимизированная статистика: число переходов, цели знакомства)
  2. Без передачи:
    • Банкам, государственным органам (кроме случаев, предусмотренных законом)

6. Права пользователя

Пользователь может:

  1. Отозвать согласие на обработку, написав администратору (@qconnect_bot)
  2. Запросить удаление своих данных (кроме логов, требующихся по 152-ФЗ)
  3. Отказаться от рекламных рассылок (кнопка «Отменить» в Боте)

Privacy Policy and Personal Data Processing Agreement

(Complies with Russian Federal Law 152-FZ "On Personal Data" and GDPR)

1. General Provisions

  1. This document defines the procedure for processing personal data of users of the QRConnect bot (hereinafter referred to as the "Bot") developed in the Telegram messenger.
  2. Using the Bot means the user's unconditional consent to the terms of this Policy.
  3. All data is processed in accordance with Russian Federal Law 152-FZ, GDPR (for EU users) and the operator's internal regulations.

2. Collected Data

The Bot processes the following personal data:

  • Name (letters only, up to 15 characters)
  • Age (from 18 to 99 years)
  • Dating purposes (selection from suggested options)
  • Contact details: phone number or Telegram @username
  • Free text (up to 1000 characters, e.g. social media links)
  • Photos/videos (formats: JPEG, PNG, MP4, MOV; video duration - up to 60 seconds)
  • Technical data: IP address, device type (iOS/Android), action logs

3. Data Processing Purposes

  1. Data is used for:
    • Creating profile cards
    • Sharing profiles with other users (third parties) only after user consent
    • Statistical analysis (devices, dating purposes, ad clicks)
    • Sending promotional materials (unless the user opts out)
  2. Files (photos/videos) are stored temporarily: after profile sharing they are deleted from the server, remaining only in the recipient's chat.

4. Data Storage and Protection

  1. Storage period:
    • Action logs — 8 months (after archiving)
    • User profiles — until deleted by the profile owner or administrator
  2. Data is protected by:
    • Encryption of transmitted information (HTTPS, TLS)
    • Restricted access (only administrator and profile owner)

5. Third-Party Sharing

  1. Data is shared only with:
    • The owner of the profile card with whom the user interacted
    • Advertising partners (anonymous statistics: number of clicks, dating purposes)
  2. No sharing with:
    • Banks, government agencies (except when required by law)

6. User Rights

The user can:

  1. Withdraw consent for processing by contacting the administrator (@qconnect_bot)
  2. Request deletion of their data (except logs required by 152-FZ)
  3. Opt out of promotional emails ("Cancel" button in the Bot)
